## Run Locally

Clone the project
```bash
  git clone https://github.com/alvs-tora/CirEx.git
```


Go to the project directory

```bash
  cd CirEx
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Run the program

```bash
  python main.py
```
